## ✨ 项目简介

css-design-skill 是一个面向**周生生组件库使用者**的前端开发 Skill，兼容各类 AI 编程助手使用。

**它的目标是**：帮助开发者在 React / Vue 项目中快速完成组件选型、页面搭建、代码生成与样式规范落地，产出品牌一致、交互稳定、可维护的前端实现。

---

## 🚀 快速开始

### 方式一：复制到你的项目中

```bash
git clone https://github.com/your-repo/css-design-skill.git
```

在 AI 编程助手中使用此项目，AI 会自动读取 `./css-design-skill/SKLL.md` 文件中的组件规范和开发指南。

### 方式二：作为 AI Skill 引用

将 `SKLL.md` 文件内容添加到 AI 助手的上下文或知识库中，即可在开发过程中使用周生生组件库的规范和组件选型建议。

---

## 🧩 组件范围

涵盖 **40+** 组件，分为五大类：

- 📝 **基础输入** - Input、Select、DatePicker、Checkbox/Radio 等
- 🎨 **布局系统** - Flex、Grid、Layout 等
- 📊 **展示导航** - Table、Tabs、Breadcrumb、Steps 等
- 💬 **反馈浮层** - Modal、Toast、Tooltip 等
- 🧩 **业务组件** - Form（配置驱动）、Carousel 等

详细组件列表和规范见 `SKLL.md`。
